# Imperium Academy Site

Este repositório contém o código fonte para o site da **Imperium Academy**.

### Sobre o Projeto

O objetivo deste projeto é desenvolver um site institucional moderno e responsivo para a Imperium Academy. A aplicação será construída com [Next.js](https://nextjs.org/) e TypeScript, utilizando React para a interface de usuário e estilos modernos (por exemplo, Tailwind CSS ou CSS Modules).

### Stack e Funcionalidades

- **Framework**: Next.js (aplicação server-side e static generation).
- **Linguagem**: TypeScript / JavaScript.
- **Estrutura de Pastas**:
  - `src/` – Contém os componentes, páginas e utilidades da aplicação.
  - `package.json` – Define dependências e scripts de build (`npm run dev` para desenvolvimento).
- **Objetivo**: Exibir informações sobre cursos, professores e matrícula, além de servir como ponto de entrada para futuros módulos educacionais.

### Desenvolvimento

Para rodar localmente:

```bash
git clone https://github.com/imperium-academy/imperium-academy-site.git
cd imperium-academy-site
npm install
npm run dev
```

Visite `http://localhost:3000` no navegador para visualizar o site em desenvolvimento.
